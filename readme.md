# FIFA 2026 Final 32 Route Map

This folder is a standalone static site.

## Share Online

### Netlify Drop
1. Go to https://app.netlify.com/drop
2. Drag this entire folder onto the page: shareable_site
3. Netlify will give you a public URL.

### GitHub Pages
1. Create a new GitHub repository.
2. Upload the contents of this folder so index.html is at the repository root.
3. In Settings > Pages, publish from the main branch root.

### Vercel
1. Create a new project.
2. Import a repo containing this folder's contents.
3. No build command is needed.

The site is self-contained: the dataset and map geometry are embedded in index.html.
